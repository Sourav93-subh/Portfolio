Projects

Here’s a quick look at some of the exciting projects I’ve worked on. These projects showcase my skills in web development, machine learning, and robotics.

Credit Card Fraud Detection (2023)

Developed a fraud detection system using Python, focusing on data preprocessing, feature engineering, and model evaluation. The project applies principles of scalable data mining and machine learning to detect fraudulent transactions in real-time.

  • Technologies Used: Python, scikit-learn, Pandas, NumPy
  • Key Skills: Data Preprocessing, Feature Engineering, Model Building, Evaluation Techniques

Plant Leaf Disease Detection (2024)

Developed a web-based application using React.js and Flask to help users upload plant leaf images and receive real-time disease diagnosis. The application integrates machine learning models trained to detect common plant diseases based on visual symptoms.

  • Technologies Used: React.js, Flask, TensorFlow, scikit-learn
  • Key Skills: Web Development, Machine Learning, Image Classification, REST APIs

Robotics Simulation Platform (2024)

Led the development of a robotic simulation platform as part of the Robotics Club at VIT Bhopal. The platform integrates various algorithms to simulate robotic movements and optimize task execution in a virtual environment.

  • Technologies Used: Python, ROS (Robot Operating System), C++
  • Key Skills: Robotics, Simulation, Algorithm Design, Distributed Computing
